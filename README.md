# GlitchGrid
Game developed during HackUPC2017 by Martí Lloveras Rosales (me)

Once you download, you can play by executing the .exe file.
You can also play it in a browser at https://martiruhay.github.io/GlitchGrid/

How to play:
- Arrows to move.
- Ctrl+Arrows to swap cells in the arrow direction.
- R to regenerate the map.
